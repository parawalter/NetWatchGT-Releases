# 📥 NetWatchGT Monitor (Releases)

¡Bienvenido al repositorio de descargas oficiales de **NetWatchGT Monitor**! 🚀

Esta es una poderosa herramienta empresarial de monitoreo de red (Ping/Latencia) que opera en segundo plano para garantizar la estabilidad de los enlaces y servidores críticos.

> [!NOTE]
> **Repositorio exclusivo para Usuarios Finales**
> Aquí se publica siempre la versión compilada más reciente de la herramienta. No encontrarás código fuente en este espacio, garantizando máxima seguridad.

---

## ⬇️ ¿Cómo instalo el programa?

Solamente necesitas **un solo archivo** para empezar a trabajar.

1. Ve a la parte superior de los archivos de este repositorio o presiona directamente el siguiente enlace.
2. Descarga el archivo: **[`NetWatchGT.exe`](https://github.com/parawalter/NetWatchGT-Releases/raw/main/NetWatchGT.exe)**.
3. Colócalo en una carpeta dedicada en tu computadora (por ejemplo: `C:\NetWatchGT\`).
4. Haz doble clic sobre él para iniciar. 

> [!IMPORTANT]  
> **No necesitas descargar ni modificar el archivo `version.json`.**
> Ese archivo (`version.json`) es de uso exclusivo del motor interno (OTA) de NetWatchGT para saber cuándo avisarte que hay una actualización disponible. Como usuario, ignóralo por completo.

## 🛠️ Instrucciones de Uso y Configuración

Al arrancar el programa `NetWatchGT.exe` por **primera vez**, el sistema generará automáticamente dos archivos de texto que podrás editar:

*   **`hosts.txt`**: Aquí ingresarás las direcciones IP o Nombres de Dominio de lo que quieres supervisar.
*   **`config.txt`**: Aquí puedes configurar tus correos electrónicos de alerta, cambiar el intervalo de monitoreo, umbrales de reintento, y personalizar el comportamiento visual.

*(Nota: Después de editar los archivos, guarda los cambios y haz clic derecho en el fondo de la aplicación para seleccionar "Recargar Configuración").*

## 🔄 Ejecución en Segundo Plano (System Tray)

NetWatchGT está diseñado para no interrumpir tu trabajo diario.

Si presionas la **"X"** (Cerrar) en la esquina superior derecha, **el programa no se apagará**. En su lugar, se camuflará en la Bandeja del Sistema de Windows (junto al reloj) y seguirá vigilando tu red.
- Para volver a ver las gráficas y la tabla: haz clic en el ícono del monitorcito azul en la barra de tareas y selecciona **Restaurar**.
- Para apagar el motor y salir de forma definitiva: selecciona **Salir Completamente**.

## 🚀 Actualizaciones Automáticas

No tienes que preocuparte por visitar esta página constantemente. NetWatchGT buscará solo sus actualizaciones y te lanzará una alerta en tu computadora para descargar la nueva versión automáticamente cuando esté lista.
